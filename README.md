RoomCheck Upload

Secure Room Photo Upload Web App | HTML, JavaScript, AWS Lambda & S3

RoomCheck Upload is a lightweight web application that allows users to securely upload room photos to AWS S3. Built with a responsive front-end in HTML/CSS/JavaScript and a serverless backend using AWS Lambda, it demonstrates a complete client-server workflow, including dynamic pre-signed URL generation for secure, direct S3 uploads.
